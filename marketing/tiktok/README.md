# Pack TikTok FitPulse (10 videos)

Ce dossier contient 10 vidéos TikTok prêtes à produire.

## Structure
- `videos/video-XX-*.md`: script complet (hook, plan, texte écran, voix off, CTA)
- `videos/video-XX-*.srt`: sous-titres prêts à importer
- `videos/video-XX-*.txt`: légende + hashtags
- `production-checklist.md`: checklist tournage

## Utilisation
1. Ouvre un `video-XX-*.md` dans Cursor.
2. Tourne les plans indiqués.
3. Monte en 9:16 (20-30s).
4. Importe le `.srt`.
5. Copie la légende du `.txt`.

CTA standard: `Lien en bio pour faire ton plan`.
